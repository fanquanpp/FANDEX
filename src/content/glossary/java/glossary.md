## 名词列表

Java 术语表已完整补全，分为三个独立文件：

### 核心语言术语（Core Language）

详见：[java-glossary-core.md](java-glossary-core.md)

包含 Java 基础概念：类与对象、继承与多态、封装、异常处理、修饰符、接口与抽象类、内部类等核心语言特性。

### 标准库术语（Standard Library）

详见：[java-glossary-stdlib.md](java-glossary-stdlib.md)

包含 Java 标准库核心类：集合框架（List/Set/Map）、I/O 流、并发编程、Stream API、Optional、线程池等常用 API。

### 高级特性术语（Advanced Features）

详见：[java-glossary-advanced.md](java-glossary-advanced.md)

包含 Java 高级特性：泛型、注解与反射、类加载机制、JVM、动态代理、模块系统、Lambda 表达式、Stream 高级用法等。

## 术语导航

### A-C

- abstract（抽象类/方法）
- annotation（注解）
- API（应用程序接口）
- array（数组）
- autoboxing（自动装箱）

### C-E

- class（类）
- Collections（集合框架）
- comparator（比较器）
- constructor（构造器）
- encapsulation（封装）

### E-G

- exception（异常）
- extends（继承）
- final（终态）
- garbage collection（垃圾回收）
- generic（泛型）

### I-M

- interface（接口）
- iterator（迭代器）
- JVM（Java 虚拟机）
- lambda（Lambda 表达式）
- method（方法）

### M-P

- multithreading（多线程）
- object（对象）
- OOP（面向对象编程）
- override（重写）
- overload（重载）

### P-R

- package（包）
- polymorphism（多态）
- reflection（反射）
- serializable（序列化）
- static（静态）

### S-Z

- stream（流）
- super（父类引用）
- this（当前对象）
- thread（线程）
- thread pool（线程池）
